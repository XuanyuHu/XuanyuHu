## 😎 About me 😎

I am Xuanyu Hu, a Senior Platform Engineer at Evernorth.

With over a decade of full-stack development experience, I specialize in Healthcare Electronic Data Interchange (EDI) and Single Sign-On (SSO) Integration. My expertise spans both backend and frontend realms, enabling me to build comprehensive and robust solutions.

In the backend domain, I am adept with Node.js, Express, Java Spring Boot, and Ruby on Rails. My proficiency extends to database technologies such as MySQL, Redis, and MongoDB, underpinning my strong backend development skills.

On the frontend, I leverage JavaScript frameworks like React.js and Next.js, and I am skilled in TypeScript. My proficiency in CSS frameworks, including Tailwind CSS, Bootstrap, and Material UI, enables me to create engaging and responsive user interfaces.

Beyond development, I have a deep understanding of secure authentication protocols and healthcare data standards. My DevOps acumen is evident in my routine use of tools like Docker, GitHub Actions, and Jenkins, coupled with cloud services experience in AWS.
